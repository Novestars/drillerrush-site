# Driller Rush Site

Static support and privacy pages for Driller Rush.

## App Store Connect URLs

After GitHub Pages is enabled for this repository, use:

- Support URL: `https://novestars.github.io/drillerrush-site/support.html`
- Privacy Policy URL: `https://novestars.github.io/drillerrush-site/privacy.html`

## Pages Setup

Repository settings:

- Settings -> Pages
- Source: Deploy from a branch
- Branch: main
- Folder: /root
